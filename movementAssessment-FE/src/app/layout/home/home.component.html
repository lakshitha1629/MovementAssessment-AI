Our Research Project Steps

In order to detect fidgety movements, identification of circular movements is important. This is the first recurrent neural network based machine learning model to identify fidgety movements using circular movements while capturing movement data using the Kinect Xbox 360 sensor. The classification is carried out using features that are based on the rotations of joints. As the performance measurements of the model using adults is quite high, this model is expected to perform in the same manner with actual infant data.This model provides promising results over 90% sensitivity with more than 85% specificity and precision when considering either rotations or coordinates. It proves that our model can be used to identify Cerebral Palsy for further research.
